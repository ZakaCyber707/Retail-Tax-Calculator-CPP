<h1 align="center">Sales Calculator (C++)</h1>

<p align="center">
A simple C++ console-based application that calculates sales totals, applies conditions, and demonstrates basic programming logic.
</p>

---

## 📌 Features

- Calculates total sales based on user input  
- Applies conditional logic (discounts, categories, etc.)  
- Uses branching structures (if-else / switch)  
- Beginner-friendly code structure  

---

## 🛠️ Technologies Used

Language: C++

IDE Used: Visual Studio 2022

Difficulty Level: Beginner

Concepts: float precision, constant tax rates, and console output.  

---

🚀 How to Run
Ensure a C++ compiler (GCC or MSVC) is installed on your system.

Clone the repository and locate sales_calculator.cpp.

Compile using g++ sales_calculator.cpp -o sales_calculator.

Execute the program to view the generated tax summary.
